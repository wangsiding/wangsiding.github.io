    Linux常用的命令

    mv

    mv 1.cpp 2.c

    把1.cpp移动并且改名为2.c

    mv 1.cpp Day2

    把1.cpp移动到Day2中去

    mv 1.c .. /Day3

    把1.c移动到父目录并且改名为Day3

    ls

    查看当面目录下的文件

    pwd

    查看当前目录

    cd

    切换目录

    ls -a

    显示当前目录下的隐藏文件

    ls-l

    显示当前文件的属性

    vi

    vi 1.txt 创建一个文件

    vi编辑器主要用来学习C语言

    IDE是集成开发软件

    mkdir

    mkdir C创建一个名字为C的文件夹/目录

    history

    显示历史命令记录

    rm

    rm 1.txt 删除文件

    rm -rf

    rm -rf C 删除文件夹/目录

    gcc xxx.c -o xxx

    编译可执行的文件并生成xxx文件

    C扩展

    include

    stdio.h 就可以用printf ，scanf

    std是命令空间，防止名字冲突

    ​ 两个max_number就会变成 wed:: max_number tied::max_number

    io 输入输出

    Linux文件系统

    .

    代表当前文件夹/目录下

    ..

    代表上一层目录

    .开头的文件

    是隐藏文件

    Git

    git init

    在当前目录下创建了一个空的git并且是隐藏文件

    管理当前目录下的文件/目录文件夹的变化

    git status

    查看当前目录下的文件/目录有无变化

    git add

    将修改后的提交到暂缓区

    git commit

    git commit -m”xxxx”

    提交并且记录当前的操作信息

    git log

    记录相关日志
